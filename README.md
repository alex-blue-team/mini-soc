# Mini-SIEM v6

A simple Python-based Mini-SIEM project created for learning cybersecurity, log analysis, event correlation, and security alert generation.

## Overview

This version processes log files, identifies failed login events, correlates them by source IP address, and generates structured security findings with severity ratings.
