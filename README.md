# mini-soc

Simple educational Mini-SOC project for log analysis.

## Description

This project reads log files and processes them line by line using a generator.

Current features:
- log file reader (generator-based)
- simple test script
- sample log data

## Files

- parser.py — log file reader
- test_parser.py — simple test script
- test.log — sample log file

## Usage

Run test script:

```bash
python test_parser.py
